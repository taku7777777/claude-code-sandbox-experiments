このセッションは sandbox 有効(.claude/settings.json の sandbox.enabled=true)で動いている前提です。network.allowedDomains は ["*.example.com"](広域ワイルドカード)、deniedDomains は ["www.example.com"](特定ホスト)です。

以下の Bash コマンドを1つだけ実行し、「承認を求められたか / 実行できたか(NETMARKER.txt が出来たか)」を1行で報告してください。ブロックされても他のツールで代替せず、結果だけ報告してください。

1. curl -sS --max-time 15 https://www.example.com -o nettest.out && echo OK > NETMARKER.txt

(www.example.com は広域ワイルドカード allow にマッチしますが、deniedDomains の特定ホスト指定が優先して遮断され、NETMARKER.txt は出来ないはずです＝「denied が広域 allow に勝つ」が観察対象です。c2-wildcard-allow と対で見ると deny の効果だけが分離できます。)
最後に、この検証で作った nettest.out と、もし出来ていれば NETMARKER.txt を削除してください。
