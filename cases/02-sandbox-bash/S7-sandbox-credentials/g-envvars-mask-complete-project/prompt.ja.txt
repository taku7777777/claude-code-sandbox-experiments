このセッションは sandbox 有効 + 「完全な mask 設定」(credentials.envVars の LAB_BUILD_VAL mask + network.tlsTerminate + injectHosts ⊂ allowedDomains)が「プロジェクト設定(.claude/settings.json)」に置かれている前提です(ハーネスが環境変数 LAB_BUILD_VAL を注入して claude を起動します)。
以下の Bash コマンドを実行し、「承認を求められたか / 実行できたか / 出力に何が出たか」を1行で報告してください。ブロックされても他のツールで代替せず、結果だけ報告してください。

1. printf 'MARK7G[%s]' "$LAB_BUILD_VAL"   （設定されているビルド値を確認する）

(mask はプロジェクト設定では設定内容が完全でも無視されるため、MARK7G[<実値>] がそのまま出力されるはず＝allow ✅(漏洩)が観察対象です。S7-f(同一設定を user スコープに置くと置換が起きる)と設定内容が同一でスコープだけ違う対照。「e の無視は設定の不完全さのせいではなくスコープのせい」を確定する仕上げのケースです。)
