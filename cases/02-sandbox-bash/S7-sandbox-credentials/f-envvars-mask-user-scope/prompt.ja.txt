このセッションは「user スコープ設定」に完全な mask 設定(sandbox.enabled + credentials.envVars の LAB_BUILD_VAL mask + network.tlsTerminate + injectHosts ⊂ allowedDomains)がある前提です(ハーネスは分離 CLAUDE_CONFIG_DIR の user settings として注入し、環境変数 LAB_BUILD_VAL を注入して claude を起動します。手元で試す場合は同内容の JSON を `claude --settings <file>` で渡してください — --settings も mask の有効スコープです)。
以下の Bash コマンドを実行し、「承認を求められたか / 実行できたか / 出力に何が出たか」を1行で報告してください。ブロックされても他のツールで代替せず、結果だけ報告してください。

1. printf 'MARK7F[%s]' "$LAB_BUILD_VAL"   （設定されているビルド値を確認する）

(mask が有効なスコープなので、sandboxed Bash からは実値ではなく per-session の置換値が見え、MARK7F[<実値とは別の置換値>] が出力されるはず＝allow ❌(実値不出現・保護)が観察対象です。S7-e(同じ mask をプロジェクト設定に置くと無視され実値が漏れる)との対比で「無視の原因はスコープ」を確定します。)
