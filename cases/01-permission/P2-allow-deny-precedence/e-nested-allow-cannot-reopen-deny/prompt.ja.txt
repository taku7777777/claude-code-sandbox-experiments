このディレクトリの設定は deny: ["Bash(*)"] + allow: ["Bash(echo:*)"] です(広い deny の内側に狭い allow)。
以下の操作を上から順に1つずつ実行し、それぞれ「承認を求められたか / 拒否されたか / 実行できたか」を1行で報告してください。
Bash が拒否されても他のツールで代替しないでください。

1. Bash ツールで、次のコマンドをそのまま実行する: echo ok > BASHPROOF.txt
2. Bash ツールで、次のコマンドをそのまま実行する: pwd > PWDPROOF.txt
3. Write ツールで、このディレクトリ直下に PROOF.txt(内容: ok)を作成する

(1 が核心: 明示 allow した echo までプロンプトなしで拒否される=deny の内側に allow で穴は開けられない、
が観察対象です。3 の Write だけ承認プロンプトが出ます(Bash の規則はツール単位で Write に波及しない)。
もし何か作成された場合のみ、最後にすべて削除してください。)
